# Priority Pages

## Tier 1

- `/` owns the broadest commercial-information hybrid cluster: Quran transliteration, Quran transliteration with Tajweed, and read Quran transliteration online.
- `/page/5` is the cleanest start point for reading because it begins at Al-Fatiha. It should remain the main internal reading CTA.
- `/page/[number]` is the core product template and now needs to accumulate better query coverage through stronger surrounding copy and future surah hubs.
- `/donate` supports the support-intent cluster and reinforces the free, community-funded positioning.

## Tier 2

- `/page/1` through `/page/4` serve as the Tajweed introduction sequence, but they are not yet strong enough to own the broader `tajweed colour guide` cluster on their own.
- Future surah hub pages should become the canonical targets for surah-name transliteration searches instead of relying on raw page routes.

## Right-To-Exist Decisions

- Keep `/` indexable and content-rich.
- Keep `/donate` indexable because it has a distinct support intent.
- Keep `/page/[number]` indexable because each page serves a unique reading job and page-specific navigation intent.
- Keep `/page/1` through `/page/4` indexable for now, but create a dedicated Tajweed guide hub later to reduce reliance on thin intro pages.
- Do not create programmatic surah/juz pages until each page has unique explanatory content and a clear internal-link role.
