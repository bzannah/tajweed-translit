## Content Checklist (Per Article)

Before publishing any article, verify:

- [ ] Unique, keyword-rich title (< 60 chars)
- [ ] Meta description (150–160 chars, includes primary keyword)
- [ ] H1 contains primary keyword
- [ ] Internal links to at least 2 other relevant pages
- [ ] External links to authoritative sources (quran.com, hadith sources)
- [ ] FAQ section with FAQPage JSON-LD schema
- [ ] Table of contents (for articles > 1,500 words)
- [ ] Featured image with descriptive alt text
- [ ] Social share image (1200x630)
- [ ] Canonical URL
- [ ] Slug contains target keyword
