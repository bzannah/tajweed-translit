# Sacred Content Rules

This project handles Quranic text — sacred scripture for Muslims worldwide.

- NEVER alter, truncate, reorder, or omit any Quranic content
- NEVER modify source PDF files in content/pdfs/
- Ensure page numbering matches the source material exactly (1-indexed, 1-1275)
- Verify surah names, verse counts, and page mappings against established sources
- Tajweed colour assignments must match the physical Tajweed Mushaf exactly
- Use respectful, neutral language in comments, logs, and error messages
- When displaying content, always preserve the original formatting and structure
- If uncertain about a data value, flag it for manual verification rather than guessing
